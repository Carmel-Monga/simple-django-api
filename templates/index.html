API Endpoints


1.

/api/apps/: List or create apps (GET returns list, POST to create).

 localhost:8000/api/apps
2.

/api/apps/<id>/: Retrieve, update or delete a single app by ID.

 localhost:8000/api/apps/1
3.

/api/apps/avg_rating_by_genre/: Average rating grouped by genre.

 localhost:8000/api/apps/avg_rating_by_genre/
4.

/api/apps/search_by_name/: Search apps using query param ?q=substring.

 localhost:8000/api/apps/search_by_name/?q=art
5.

/api/apps/category_stats/: Counts and average rating per category.

 localhost:8000/api/apps/category_stats/
6.

/api/reviews/by_sentiment/: Filter reviews by ?sentiment=positive|negative|neutral.

 localhost:8000/api/reviews/by_sentiment/?sentiment=positive
